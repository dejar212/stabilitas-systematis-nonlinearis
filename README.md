# Качественный анализ областей устойчивости нелинейной системы управления методом функций Ляпунова

Подготовил: Сороколетов Денис Сергеевич

Работа посвящена глубокому анализу устойчивости нелинейных динамических с-м с использованием классических методов теории управления

Основное внимание уделено построению областей притяжения и анализу робастности с-мы при изменении физических параметров

## Описание проекта

В рамках работы решаются следующие задачи
1 Вывод нелинейных ур-й движения с использованием формализма Лагранжа
2 Линеаризация с-мы и проверка условий устойчивости по первому приближению
3 Построение ф-ции Ляпунова на основе полной энергии с-мы для док-ва асимптотической устойчивости
4 Построение фазовых портретов для визуализации траекторий в окрестности равновесия

## Математический аппарат

Для анализа используется ф-ция Ляпунова вида
\[ V(x) = T + \Delta V \]

Здесь $T$ это кинетическая энергия а $\Delta V$ это потенциальная энергия отн-но положения равновесия

Моменты инерции звеньев рассчитываются по ф-ле
\[ I = \frac{mL^2}{12} \]

Условие устойчивости проверяется через производную ф-ции Ляпунова
\[ \dot{V}(x) \le 0 \]

## Исследование робастности

В работе проведено исследование влияния неопределенности параметров на критический угол отклонения при котором с-ма сохраняет устойчивость
